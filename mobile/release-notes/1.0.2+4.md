# 1.0.2+4

## 새 기능
- 행사 현황 조회 기간을 달력에서 시작일~종료일 범위로 한 번에 선택할 수 있습니다.
- 클레임 현황도 동일한 범위 달력(최대 7일)으로 기간을 선택하며, 선택할 수 없는 날짜는 취소선으로 표시됩니다.
- 주문 화면에서 거래처를 선택 바텀시트로 더 편하게 조회·선택할 수 있습니다.
- 클레임 현황 목록을 20건씩 페이지로 나눠 보여주고, 상세 화면에서 사진과 전체 항목을 확인할 수 있습니다.
- 사이드 메뉴 하단에서 현재 앱 버전을 확인할 수 있습니다.

## 개선
- 행사 현황·클레임 현황·점검 조회에서 검색 시 화면이 비거나 멈추던 문제를 수정했습니다.
- 로그인 실패 시 화면 이동 없이 입력 화면에서 바로 오류 안내가 표시됩니다.
- 클레임 등록 시 불량 내역·수량 입력 중 커서가 튕기던 문제를 수정했습니다.
- 유통기한 목록의 거래처 선택 목록이 비어 보이던 문제를 수정했습니다.
- 현장 점검·클레임 등 기간 조회를 최대 7일로 정확히 맞췄습니다.
- 조장 홈의 팀 출근 현황과 여사원 일별 현황 표시를 정비했습니다.
- 앱 아이콘을 오뚜기 공식 아이콘으로 변경했습니다.
