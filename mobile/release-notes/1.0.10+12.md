# 1.0.10+12

## 새 기능
- 제품 추가 화면에서 검색 결과를 끝까지 이어 볼 수 있고, 전체 건수가 표시됩니다.
- 자동 로그인을 끈 경우, 앱을 다시 열었을 때 일정 시간이 지나면 자동으로 로그아웃됩니다.

## 개선
- 제품 추가 화면에서 소분류를 중분류 선택 없이도 바로 선택할 수 있습니다.
- 제품 추가 화면의 제품명·바코드 입력란 디자인을 정리했습니다.
- 주문 화면에서 금액·수량이 잘못 표시되던 문제를 수정했습니다.
- 푸시 알림이 정상적으로 수신되도록 안정화했습니다. (iOS 최신 버전 포함)
- 내 일정 캘린더 화면의 하단 표시 여백을 정리했습니다.
- 세션 만료 안내가 반복해서 나타나던 문제를 수정했습니다.
- 자동 로그인 실패 시 로그인 화면이 중복으로 뜨던 문제를 수정했습니다.
