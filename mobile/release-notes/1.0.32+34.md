# 1.0.32+34

## 개선
- 클레임·제안·주문 화면에서 전송 및 승인 요청을 바로 할 수 있습니다.
- 출근 등록 시 위치를 가져오지 못하면 원인별로 안내 메시지를 보여줍니다.
- 제품 클레임에서 영수증 사진, 구매 금액, 구매 방법, 요청 사항을 자유롭게 삭제·수정할 수 있습니다.
- 클레임·현장 점검 사진 업로드 안정성을 높였습니다. (이미지 형식 문제로 등록이 거부되던 현상 해결)
- 주문 현황 등 화면의 글자·아이콘 색상을 더 잘 보이도록 개선했습니다.
- 전산 매출에서 제품을 선택할 때 앱이 종료되던 문제를 수정했습니다.
