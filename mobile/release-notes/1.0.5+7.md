# 1.0.5+7

## 개선
- 주문 작성 화면에서 바코드 스캔과 제품 검색으로 품목을 빠르게 담을 수 있습니다. (전용·시식 품목은 자동 차단)
- 다른 기기에서 로그인하면 기존 기기는 즉시 로그아웃되며, 사유를 안내합니다.
- 로그인 직후 스플래시 화면이 간헐적으로 멈추던 문제를 해결했습니다.
- 로그아웃 후 홈 화면이 빈 채로 계속 로딩되던 문제를 해결했습니다.
- 자동 로그인 갱신 중 네트워크 요청이 끊겨도 로그인이 풀리지 않도록 안정성을 높였습니다.
- 행사 매출 현황의 필터 순서를 거래처 → 기간 → 행사명으로 정리했습니다.
- 제안하기 등록 시 발생하던 오류를 수정했습니다.
