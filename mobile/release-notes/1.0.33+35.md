# 1.0.33+35

## 새 기능
- 대리출근을 지점을 선택해 전체 거래처에서 등록할 수 있습니다.
- 주문서 작성이 더 편해졌습니다. 담기 버튼 상태와 납기일을 바로 선택할 수 있습니다.
- 여사원 일별현황에서 항목을 바로가기로 눌러 이동하고, 관련 수치를 함께 확인할 수 있습니다.
- 주문에서 특정 거래처 전용상품을 구분해 표시합니다.
- 안전점검 체크리스트가 처음부터 펼쳐진 상태로 표시됩니다.

## 개선
- 월간 캘린더의 안전점검·보고완료·총건수 표시를 정확하게 맞췄습니다.
- 주문 제품은 최대 100개까지 담을 수 있도록 미리 안내합니다.
- 출근등록 화면에서 고정 근무자 등록 버튼을 거래처 목록 바로 아래로 옮겼습니다.
- 주문 등록 후 다시 들어갔을 때 이전에 작성하던 내용이 남지 않도록 초기화합니다.
- 주문 등록 중 오류가 났을 때 어느 단계(재고/여신 조회)에서 실패했는지 함께 표시합니다.
