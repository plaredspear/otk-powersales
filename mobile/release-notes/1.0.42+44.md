# 1.0.42+44

## 개선
- 주문 제품추가에 주문이력 탭이 추가되어, 거래처의 최근 주문 제품을 기간별로 조회해 담을 수 있습니다.
- 주문 상태별 의미를 확인할 수 있는 상태 안내 화면이 추가되었습니다.
- 제품추가 즐겨찾기 탭에서 키워드로 목록을 검색할 수 있습니다.
- 제품검색에서 중분류/소분류만 선택해도 바로 검색됩니다.
- 주문 상태 표기를 정비했습니다 — 승인완료는 "전송완료"로, 주문취소는 "주문취소요청완료"로 표시되며 처리 지연 안내가 추가되었습니다.
- 주문상세의 재전송 버튼을 제거하고 재주문 안내 문구를 추가했으며, 반려제품 섹션에 처리 지연 안내를 추가했습니다.
- 비밀번호 변경을 마이페이지에서 한 화면으로 통합하고 입력 화면을 개선했습니다.
- 로그아웃 메뉴를 마이페이지 그룹으로 이동하고 정상 동작하도록 수정했습니다.
- 소비기한 수정 화면에서 변경 사항이 없으면 저장 버튼이 비활성화됩니다.
- 화면을 다시 열었을 때 임시저장 복원 안내가 뜨지 않던 문제를 수정했습니다.
