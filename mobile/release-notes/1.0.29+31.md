# 1.0.29+31

## 새 기능
- 홈 화면 일정 카드가 근무형태(행사·판촉 등)별로 구분되어 표시됩니다.
- 홈 일정 카드에 하루 일정이 최대 3개까지 항상 정렬되어 보입니다.
- 교육 메뉴의 라벨·아이콘·이동 링크를 정비했습니다.
- 비밀번호 변경 화면에서 설정 가능한 비밀번호 조건을 안내합니다.

## 개선
- 여사원 일별 현황에서 바로가기 스크롤이 정상 동작하도록 복구하고, 일정 유형별 색상을 구분했습니다.
- 조장 대리 출근을 여러 건 등록할 때 일부만 실패해도 나머지가 정상 처리됩니다.
- 내 정보에서 권한이 지정되지 않은 경우 표시를 '-'로 정리했습니다.
