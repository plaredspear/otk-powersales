# 1.0.8+10

## 새 기능
- 앱 전체 글꼴이 오뚜기 전용 서체로 바뀌어 더 또렷하게 보입니다.
- 현장점검 등록 화면이 개선되고, 작성 중인 내용을 서버에 임시저장할 수 있습니다.
- 행사 상세의 '진행율'이 '기간 경과율'로 바뀌고 남은 일수를 함께 보여줍니다.
- 앱 업데이트가 있을 때 설치 안내가 더 친절해졌습니다.

## 개선
- 현장점검·일매출 사진의 보안이 강화되고, 등록·상세 화면 구성을 정비했습니다.
- 종료된 행사의 기간 경과율이 정확히 100%로 표시됩니다.
- 클레임 등록 화면을 다시 열었을 때 이전 입력값이 남던 문제를 고쳤습니다.
- 점검 등록 화면을 다시 열었을 때 이전 테마·사진이 남던 문제를 고쳤습니다.
- 홈 화면의 여백·정렬·그림자 등 레이아웃을 전반적으로 정돈했습니다.
