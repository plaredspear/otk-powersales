# 1.0.18+20

## 새 기능
- 매출 조회 화면(전산/POS/물류) 진입 시 기본 거래처가 자동 선택되고 바로 조회됩니다.
- 조장 홈 빠른메뉴의 "내 일정"이 "일정 관리"(팀원 월간 일정)로 바뀌었습니다.

## 개선
- 비밀번호 확인 시 잘못 입력해도 로그아웃되지 않도록 수정했습니다.
- 물류 클레임 목록의 검색 조건이 항상 표시되도록 정리했습니다.
- 월 매출(물류) 화면의 진도율·실적 표시를 당월 기준에 맞게 정리했습니다.
- 월 매출(물류) 재진입 시 상단 거래처명이 유지됩니다.
- 주문 목록·상세의 승인상태 표기와 제품 라인 표시를 정확하게 맞췄습니다.
