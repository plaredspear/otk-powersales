# 1.0.17+19

## 새 기능
- 물류 클레임 조회에서 거래처와 등록일 기간으로 검색할 수 있습니다.

## 개선
- 주문 등록·임시저장이 실패하면 실제 원인을 화면에 표시합니다.
- 현장 점검 화면 명칭을 '현장 점검 등록'으로 통일했습니다.
- 임시저장한 주문을 다시 불러올 때 단가·입수를 최신 제품 정보로 다시 채워 소계가 정확히 계산됩니다.
- 여신 한도를 초과하면 승인 요청 단계에서 미리 안내합니다.
- 승인 요청 시 수량 검증을 총 EA 기준으로 맞춰, 박스만 입력해도 정확히 확인됩니다.
- 납기일 안내 기준을 정비했습니다.
