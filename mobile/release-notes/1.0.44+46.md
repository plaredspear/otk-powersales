# 1.0.44+46

## 새 기능
- 주문서 작성 시 추가 품목을 제품명·코드로 검색해 빠르게 찾을 수 있습니다.
- 주문 상세에서 취소·변경된 관련 주문을 함께 확인할 수 있습니다. (내 주문 / 거래처별 주문 모두)
- 현장점검에서 답변을 입력하면 다음 항목으로 자동 이동해 점검이 더 매끄러워졌습니다.

## 개선
- 주문 등록 시 알라딘 전송에 실패하거나 거부되면 실패 사유를 화면에서 바로 확인할 수 있습니다.
- 주문 취소 실패 시 실제 사유 메시지를 그대로 보여 드립니다.
- 주문 수량 단위가 맞지 않을 때 안내 문구를 더 이해하기 쉽게 다듬었습니다.
- 소비기한 등록·수정 화면에서 이전에 입력한 설명이 남아 있던 문제를 해결했습니다.
