# 1.0.37+39

## 새 기능
- 주문 상세 상단에서 총 주문취소 건수를 한눈에 볼 수 있습니다.
- 여신 조회에 실패하면 바로 다시 조회할 수 있는 버튼이 추가되었습니다.
- 제품 클레임 상세에서 담당자 직위와 주문번호를 함께 확인할 수 있습니다.

## 개선
- 주문 상세의 배송 상태 표시를 정확하게 맞췄습니다.
- 주문 상태가 없는 경우 발생하던 앱 멈춤을 해결하고, 반려 수량 표시가 잘리던 문제를 개선했습니다.
- 주문 등록·취소, 여신 조회 등 서버 응답이 오래 걸리는 작업의 대기 시간을 늘려 실패로 처리되던 경우를 줄였습니다.
- 주문 등록 후 SAP 전송 결과 확인 과정을 보완해 전송 성공 여부를 더 정확하게 안내합니다.
- 클레임의 전송 상태 표시를 명확히 하고, 알라딘 전송 상태를 함께 확인할 수 있도록 했습니다.
