# 1.0.41+43

## 개선
- 거래처 주문 상세에서 배송수량을 EA 단위로 함께 확인할 수 있습니다.
- 홈 화면 출근 배지를 눌러 오늘의 등록 현황을 바로 볼 수 있습니다.
- 사진 첨부 시 남은 매수만큼 갤러리에서 여러 장을 한 번에 선택할 수 있습니다.
- 제품 검색 즐겨찾기 탭에서 전체 선택이 가능합니다.
- 주문 상세에 미납 제품 섹션이 추가되어 미납 내역을 제품 목록 최상단에서 바로 확인할 수 있습니다.
- 여사원 내 일정 캘린더가 정상적으로 표시되도록 수정했습니다.
- 여사원 내 일정 화면 진입 시 최신 내용으로 자동 갱신되며, 당겨서 새로고침을 지원합니다.
- 주문 상세의 결품 상태를 '미납' 으로 알기 쉽게 표기합니다.
- 주문 상세의 승인 금액·반려 내역·처리 현황 표시를 정확하게 맞췄습니다.
- 임시 비밀번호가 사번 기반 형식으로 변경되어 안내가 명확해졌습니다.
