# 1.0.38+40

## 개선
- 주문 현황·소비기한 관리 화면에서 검색 버튼 없이 조건을 고르면 바로 조회됩니다.
- 제품 검색 화면을 주문서와 동일한 3탭 방식으로 통일했습니다.
- 날짜 선택(주문 납기일·소비기한·물류클레임 발생일자)을 탭 한 번으로 바로 확정할 수 있습니다.
- 주문 승인·취소 화면에 처리 소요 시간 안내를 추가하고, 주문 상세를 당겨서 새로고침할 수 있습니다.
- 주문 취소 시 통신 지연으로 실패처럼 보이던 문제를 개선해, 실제 처리 결과를 다시 확인하도록 했습니다.
- 주문 취소 대기 시간을 늘려 시간이 오래 걸리는 취소도 정상 처리되도록 했습니다.
- 주문 현황 상태 필터에서 불필요한 '임시저장' 항목을 제거했습니다.
- 통신 오류 발생 시 알 수 없는 에러 문구가 그대로 노출되던 문제를 개선했습니다.
- 홈 화면을 당겨 새로고침할 때 하단에 노란색이 잠깐 보이던 문제를 수정했습니다.
